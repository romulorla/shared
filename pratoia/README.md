# PratoIA 🍽️🇧🇷

**AI-powered calorie & macro tracker built for Brazilians.**

Take a photo of your meal → get instant nutritional breakdown powered by AI vision models, using Brazilian food databases (TACO/TBCA).

## Stack
- **Frontend:** Flutter (iOS + Android)
- **Backend:** Node.js + Supabase (PostgreSQL + Auth + Storage)
- **AI:** Gemini Flash 2.0 (primary) / GPT-4.1 nano (fallback) for food recognition
- **Food DB:** TACO/TBCA + custom Brazilian dishes database

## Key Differentiators
- 🇧🇷 Built for Brazilian food (açaí, feijoada, pão de queijo, marmitex)
- 💰 Affordable pricing with PIX support
- 🗣️ Portuguese-first UX
- 📊 TACO/TBCA nutritional data (not USDA)
- 🆓 Generous free tier to fight subscription fatigue
