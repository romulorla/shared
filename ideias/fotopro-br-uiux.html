UX Principles

Design decisions guided by clarity, speed, and trust.

Speed First

Camera opens instantly. No splash screens, no tutorials on first launch. Users can capture within 2 seconds of opening the app.

One Path Forward

Linear flow: Camera, Preview + Edit, Processing, Export. Minimize decisions. Always show the next step clearly.

Show, Don't Tell

Real-time previews everywhere. Background changes update instantly. Template selections show immediate results.

Safe to Explore

Non-destructive editing. Undo always available. Originals preserved. Users should never fear losing their work.

Made for Brazil

Mercado Livre and Shopee as first-class export targets. Portuguese throughout. Currency in BRL. Local marketplace requirements built-in.

Professional Results

AI does the heavy lifting. Users get studio-quality output without studio knowledge. Smart defaults over manual controls.

Micro-interactions

Subtle animations that communicate state and delight without distraction.

Brief screen flash (white, 80% opacity) + subtle haptic. Mimics real camera feedback.

Each step fades in with a checkmark on completion. Active step pulses subtly. Progress feels alive, not stuck.

Tapped option scales slightly with border highlight in accent color. Subtle haptic confirms selection before navigating to processing.

duration: 150ms | easing: ease-out

Export Success

Checkmark draws itself in a single stroke. Subtle confetti particles (muted colors, 8 particles max). Celebratory but restrained.

checkmark: 400ms | confetti: 800ms | fade-out: 300ms

Accessibility

Inclusive design ensures the app works for everyone.

Color Contrast

All text meets WCAG 2.1 AA. Primary text on white: 14.5:1. Secondary text: 5.7:1. Accent on white: 4.6:1.

Touch Targets

Minimum 44x44pt for all interactive elements. Shutter button: 56pt. Adequate spacing between targets to prevent mis-taps.

Screen Readers

Full VoiceOver and TalkBack support. Semantic labels on all controls. Image descriptions for processed photos.

Motion Sensitivity

Respects "Reduce Motion" system setting. Animations replaced with instant state changes. No parallax effects.

Dynamic Type

Supports system font scaling. Layout adapts gracefully up to 200% text size. No text truncation on critical UI.

One-Handed Use

Key actions reachable in thumb zone. Bottom-aligned controls. No critical interactions at screen top on large devices.
